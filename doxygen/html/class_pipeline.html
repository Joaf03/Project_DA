Detailed Description

The Pipeline class represents a pipeline entity for water transport.

It stores information about a pipeline, including its source, target, capacity, and whether it is bidirectional or not.

Time Complexity:

  • Accessors (getters): O(1)
  • Mutators (setters): O(1)
  • Constructor (default and parameterized): O(1)
