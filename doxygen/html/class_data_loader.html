Detailed Description

The DataLoader class provides methods to load data from files.

It includes static methods to load water reservoirs, pumping stations, delivery sites, and pipelines from specified files.

vector< DeliverySite > DataLoader::loadDeliverySites ( string & filename)
static

Load delivery sites from a file.

Reads data from the specified CSV file and constructs DeliverySite objects. Each line in the file represents a delivery site with attributes separated by commas. The format of each line should be: city,id,code,demand,population.

Parameters
filenameThe name of the file to load delivery site data from.
Returns
A vector of DeliverySite objects loaded from the file.
