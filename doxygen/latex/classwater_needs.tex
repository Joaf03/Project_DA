\doxysubsection{Detailed Description}
The \doxylink{classwater_needs}{water\+Needs} class contains methods to determine the cities with water deficit, the maximum flow that can be sent to each or a specific delivery site, and to balance the load of the pipelines. 

Time Complexity\+:
\begin{DoxyItemize}
\item Cities with deficit\+: O(P+C) where P is the number of pipelines and C is the number of cities.
\item Maximum flow\+: O(\+R\texorpdfstring{$\ast$}{*}\+C\texorpdfstring{$\ast$}{*}\+V\texorpdfstring{$\ast$}{*}\+E\texorpdfstring{$^\wedge$}{\string^}2) where R is the number of water reservoirs, C is the number of cities, V is the number of vertices in the graph and E is the number of edges.
\item Balance load\+: O(\+R\texorpdfstring{$\ast$}{*}\+C\texorpdfstring{$\ast$}{*}\+V\texorpdfstring{$\ast$}{*}\+E\texorpdfstring{$^\wedge$}{\string^}2) where R is the number of water reservoirs, C is the number of cities, V is the number of vertices in the graph and E is the number of edges. 
\end{DoxyItemize}
