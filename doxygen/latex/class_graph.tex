\doxysubsection{Detailed Description}
The \doxylink{class_graph}{Graph} class represents a network graph consisting of water reservoirs, pumping stations, delivery sites, and pipelines. 

It stores information about water reservoirs, pumping stations, delivery sites, and pipelines, and provides methods to add and retrieve these entities from the graph.

Time Complexity\+:
\begin{DoxyItemize}
\item Adding water reservoirs\+: O(n)
\item Adding pumping stations\+: O(n)
\item Adding delivery sites\+: O(n)
\item Adding pipelines\+: O(n)
\item Getting water reservoirs\+: O(1)
\item Getting pumping stations\+: O(1)
\item Getting delivery sites\+: O(1)
\item Getting pipelines\+: O(1)
\item BFS\+: O(V+E)
\item Edmonds-\/\+Karp algorithm\+: O(\+VE\texorpdfstring{$^\wedge$}{\string^}2)
\item Determining cities affected by ruptured pipelines\+: O(\+VE\texorpdfstring{$^\wedge$}{\string^}3)
\item Balancing load\+: O(\+R\texorpdfstring{$\ast$}{*}\+C\texorpdfstring{$\ast$}{*}\+V\texorpdfstring{$\ast$}{*}\+E\texorpdfstring{$^\wedge$}{\string^}2) 
\end{DoxyItemize}
